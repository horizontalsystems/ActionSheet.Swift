# ActionSheet.Swift

Customizable replacement for UIAlertController.

## Installation

### Swift Package Manager

```swift
dependencies: [
    .package(url: "https://github.com/horizontalsystems/ActionSheet.Swift.git", .upToNextMajor(from: "1.0.0"))
]
```
